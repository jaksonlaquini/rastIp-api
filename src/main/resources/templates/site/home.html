Conheça o RastIp

O RastIp é um Sistema de Rastreio de Informações de Produtos, com objetivo de informar e facilitar ao consumidor, a origem e cultivo do produto, assim tendo uma maior confiança para adiquiri-lo.

O acesso ao RastIp é simples, basta criar um login simples e acessá-lo. O Sistema é totalmente online, não necessitando de instalações de software auxiliares, e nem de aparelhos com requisitos de configurações.

As principais funcionalidades do sistema são:

  • Cadastro de produto, produtor e endereço da produção.
  • Impressão de etiquetas, com as informações necessárias para o rastreio do produto.
  • Gerador de Qrcode, para acesso ao rastreio via smartphone.
  • Exibição das informações de origem do produto via google Maps.
